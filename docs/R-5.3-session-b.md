## Architecture GPS finale

```
MapViewModel._isRideMatched.collect { matched ->
    locationController.applyGpsMode(if (matched) HIGH else COARSE)
    context.startForegroundService / startService(RideForegroundService)
}

MapLocationController.applyGpsMode(mode)       ← provider re-config (FusedLocation)
    └─ locationServiceController.setMode(mode) ← StateFlow mise à jour

LocationServiceController.shouldAcceptLocation(loc)
    ├─ GpsMode.OFF → false (drop all)
    ├─ accuracy > 50m → false
    ├─ distanceTo(prev) < 8m → false
    └─ true → lastAcceptedLocation = loc

RideForegroundService
    ├─ startForeground(API ≥ 34: +ServiceInfo.FOREGROUND_SERVICE_TYPE_LOCATION)
    ├─ ic_notification_ride (white fill vector, 24dp)
    └─ stopWithTask="false" (GPS survit swipe-away)
```
